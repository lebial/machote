## Setup

### Prerequisites
- chruby
- ruby 2.3.4
- postgres 9.4+
- yarn 1.3.2+
- node 8.9+

### Getting the app running
1. Clone this repo
2. `cd boilerplate_app`
3. `sh bin/setup.sh`
